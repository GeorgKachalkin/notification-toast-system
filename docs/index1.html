GenKern Notice

Легковесная система уведомлений на чистом JavaScript

v1.0.0 Zero Dependencies

Типы уведомлений

Четыре основных типа для разных ситуаций: успех, ошибка, предупреждение и информация.

notice({
  type: 'success',
  title: 'Готово!',
  message: 'Настройки успешно сохранены.',
  icon: 'fas fa-check'
});

Позиционирование

Размещайте уведомления в любом углу экрана или по центру.

Top Left
Top Center
Top Right
Bottom Left
Bottom Center
Bottom Right
notice({
  message: 'Уведомление снизу слева',
  from: 'bottom',
  align: 'left'
});

Прогресс-бар

Визуальный индикатор времени до автоматического закрытия.

notice({
  type: 'info',
  message: 'Загрузка завершена',
  show_progress_bar: true,
  delay: 6000
});

Кликабельные уведомления

Сделайте уведомление ссылкой для перехода на другую страницу.

notice({
  type: 'info',
  title: 'Новое обновление',
  message: 'Доступна новая версия. Нажмите для просмотра.',
  url: 'https://github.com'
});

Дополнительные опции

Гибкая настройка под любые задачи.

// Без автозакрытия
notice({
  message: 'Важное сообщение',
  delay: 0
});

// Без кнопки закрытия
notice({
  message: 'Только для чтения',
  allow_dismiss: false
});

API Reference

Все доступные параметры функции notice().

Параметр Тип По умолчанию Описание
type string 'info' success, danger, warning, info
title string '' Заголовок уведомления
message string '' Текст сообщения
icon string '' CSS-класс иконки
delay number 5000 Время автозакрытия (мс)
from string 'top' top, bottom
align string 'right' left, right, center
show_progress_bar boolean false Показывать прогресс-бар
